# 校园失物招领平台

帮助校园师生快速找回丢失物品，传递校园正能量

## 项目简介

校园失物招领平台是一个帮助大学生和教职工找回丢失物品、归还捡到物品的互助平台。

## 功能特性

- 🔍 快速搜索失物招领信息
- 📁 分类浏览物品
- 📱 响应式设计
- 💬 评论交流功能
- 💾 本地数据存储

## 技术栈

- HTML5 + CSS3
- JavaScript (ES6+)
- LocalStorage

## 快速开始

```bash
git clone https://github.com/929930/campus-lost-found-new.git
cd campus-lost-found-new
open index.html
```
